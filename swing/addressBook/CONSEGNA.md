# Consegna
## Versione 1
Realizza una applicazione Java dotata di una apposita interfaccia grafica SWING che gestisce (inserimento, cancellazione, modifica e ricerca) una rubrica di contatti.  
Predisporre una apposita classe "Contatto" dotata degli opportuni attributi e metodi e una seconda classe "Rubrica" che gestisce con i necessari metodi un arrayList di Contatto.
## Versione 2
I contatti presenti in rubrica vengono memorizzati in un file di testo CSV, all'avvio dell'applicazione questi dati vengono recuperati dal file e inseriti nell'array alla chiusura dell'applicazione la rubrica, con gli eventuali aggiornamenti, viene nuovamente memorizzata nel file csv.
